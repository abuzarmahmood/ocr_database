Database to allow search of scanned documents.
Operate on Pareto principle (something better than nothing).

Frontend:
    - Submit scans
	- Upload scans
	- Provide metadata
	    - Document Types
    - Search scans
        - By metadata
	- By content (OCR)
	- View scans
	    - Path to saved scan

Backend:
    - Accept scans
	- Store scans
	- Extract metadata
	- Extract text from scans
	- Search scans
	    - By metadata
	    - By content
	- Return path to saved scan
